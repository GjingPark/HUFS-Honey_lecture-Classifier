# HUFS-Honey_lecture-Classifier
## 맞춤형 팀플 및 과제 부담 최소화 강의 분류 시스템
강의계획서 텍스트를 분석하여 팀플 유무와 과제 부담 지수를 예측해주는 AI 서비스입니다.
- Author : Euijin Park
- Model: Dual DistilKoBERT (Classification + Regression)
## 프로젝트 소개
### **제작 의도**
- 본 프로젝트 진행자는 한국외국어대학교 2학년 학생으로, 팀플보단 혼자 듣는 것을 선호하며 과제 또한 적은 강의
  즉, 꿀강을 원하는 학생입니다. 매번 강의 계획서를 읽어가며 꿀강인 교양, 전공선택을 찾는 것에 지친, 본인과 같은
  학생들이 많을 것으로 예상됩니다. 따라서 강의계획서를 직접 읽지 않고 복사 붙여넣기로 꿀강인지 아닌지 분류해주는
  모델이 필요, 많은 인원이 사용할 것으로 판단하여 제작을 시도해보게되었습니다.

**Features**:
- 1. 강의계획서 전체 텍스트 자동 파싱 및 추출
- 2. 팀플 위험도(%) 예측
- 3. 과제 부담 지수(0~100) 예측
- 4. 판단 근거 텍스트 제공

### 실행 방법
1) 링크 접속
   - https://hufs-honeylecture-classifier.streamlit.app/
   - 별도의 모델 가중치 다운을 받지 않아도, 이미 웹으로 구현되어 있기에 링크로 바로 접속하시면 됩니다.
2) 모델 실행
   - 분류를 원하는 강의계획서 텍스트를 전체 복사(Ctrl + A, Ctrl + C)하여 웹에 붙여넣기(Ctrl + V)를 진행합니다.
   - 강의 분류 시작 버튼을 누릅니다.
3) 수강 신청
   - 꿀강을 찾아 수강신청을 합니다.
